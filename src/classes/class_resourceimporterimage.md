# ResourceImporterImage

**继承：** [`ResourceImporter`](class_resourceimporter.md) **<** [`RefCounted`](class_refcounted.md) **<** [`Object`](class_object.md)

Imports a image for use in scripting, with no rendering capabilities.

## 描述

This importer imports [`Image`](class_image.md) resources, as opposed to [`CompressedTexture2D`](class_compressedtexture2d.md). If you need to render the image in 2D or 3D, use [`ResourceImporterTexture`](class_resourceimportertexture.md) instead.
