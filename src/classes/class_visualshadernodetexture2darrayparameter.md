# VisualShaderNodeTexture2DArrayParameter

**继承：** [`VisualShaderNodeTextureParameter`](class_visualshadernodetextureparameter.md) **<** [`VisualShaderNodeParameter`](class_visualshadernodeparameter.md) **<** [`VisualShaderNode`](class_visualshadernode.md) **<** [`Resource`](class_resource.md) **<** [`RefCounted`](class_refcounted.md) **<** [`Object`](class_object.md)

A visual shader node for shader parameter (uniform) of type [`Texture2DArray`](class_texture2darray.md).

## 描述

This parameter allows to provide a collection of textures for the shader. You can use [`VisualShaderNodeTexture2DArray`](class_visualshadernodetexture2darray.md) to extract the textures from array.
